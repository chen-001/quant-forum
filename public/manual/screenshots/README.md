## 📋 截图命名规范

1. **文件夹结构**：按章节分类存放
2. **文件名格式**：`功能名-描述.png`（小写，用连字符分隔）
3. **图片格式**：PNG（推荐）或 JPG
4. **图片尺寸**：建议宽度 1200px 左右，保持清晰
